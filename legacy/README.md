# Legacy Archive

## Meaning of `legacy`

These files are preserved because they document how the current project was
formed. They are not deleted, but they are not authoritative model
specifications. Current work must begin from `../model_notes/`.

The archive was created on 2026-06-08 during consolidation of `BEV_new_program`
into the self-contained `NEV_model_project`.

## Restoration rule

Do not move a legacy file back into the active project wholesale. Extract only
the needed argument, equation, empirical idea, or citation; compare it with the
current baseline; add it to the relevant module with an explicit status; and
record any resolved modeling choice in `99_decision_log.md`.
